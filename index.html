CSS-transition

    CSS-transition позволяет анимировать переход между различными свойствами с течением времени, анимировать можно как все так и отдельные свойства с разным промежутком времени.

    Пример записи: [transition: all 1s ease 0s;]

1)Transition-property

    Содержит название CSS-свойств, к которым будет применен эффект перехода, анимация неуказанных свойств будет происходить моментально.

1)Все свойства изменяются плавно.

2)Плавно изменяется только свойство width. 

						
.All{
	transition: all 2s;
}

.duration:hover{
	background-color: black;

	width: 100px;
	height: 100px;
}
						
					
						
.One{
	transition: width 2s;
}

.duration:hover{
	background-color: black;

	width: 100px;
	height: 100px;
}

2)Transition-duration

    Задаёт промежуток времени, в течение которого должен осуществляться переход. Если разные свойства имеют разные значения для перехода, они указываются через запятую.Если промежуток 0, то анимации не будет.

1)Длительность анимации 5 секунд.

2)Длительность анимации 0,5 секунд. 

						
.Long{
	transition: all 5s;
}

.duration:hover{
	background-color: black;

	width: 100px;
	height: 100px;
}
						
					
							
.Short{
	transition: all 0.5s;
}

.duration:hover{
	background-color: black;

	width: 100px;
	height: 100px;
}

3)Transition-timing-function

    Свойство задаёт временную функцию, которая описывает скорость перехода объекта от одного значения к другому.

1)Медленно-быстро-медленно.

2)Линейная. 

						
.Ease{
	transition: all 2s ease;
}

.duration:hover{
	background-color: black;

	width: 100px;
	height: 100px;
}
						
					
						
.Linear{
	transition: all 2s linear;
}

.duration:hover{
	background-color: black;

	width: 100px;
	height: 100px;
}

4)Transition-delay

    Необязательное свойство, позволяет сделать так, чтобы изменение свойства происходило не моментально, а с некоторой задержкой.

1)Длительность задержки 5 секунд.

2)Длительность задержки 0,5 секунд. 

						
.dLong{
	transition: all 2s ease 5s;
}

.duration:hover{
	background-color: black;

	width: 100px;
	height: 100px;
}
						
					
						
.dShort{
	transition: all 2s ease 0.5s;
}

.duration:hover{
	background-color: black;

	width: 100px;
	height: 100px;
}

5)Сокращенная запись

    Все свойства, отвечающие за изменение внешнего вида элемента, можно объединить в одно свойство transition transition: transition-property transition-duration transition-timing-function transition-delay; 

						
transition: 1s;
						
					
						
transition: all 1s ease 0s;
